# Sample Questions

These questions are useful for testing the demo knowledge base:

- What does Ollama Agent Kit do?
- How does the chat loop handle tool calls?
- Where are debug logs written?
- How does the Markdown RAG flow work?
- How do I add a file to the knowledge base?
- How do I disable automatic retrieval?
- What should I check when Ollama is unreachable?
- How does the controlled Python execution tool work?
- Show a demo where the agent uses Python to calculate a result.
- What happens if the Python tool imports `os` or runs too long?

You can also ask compound questions during a demo, for example:

- Explain the RAG workflow and show the citation format.
- Compare normal chat mode with `--no-rag` mode.
- Summarize the tool loop and the hallucination guard behavior.
- Explain the Python execution demo and describe the safety boundaries.

These questions are intentionally aligned with the corpus so retrieval can produce visible, easy-to-explain results.
